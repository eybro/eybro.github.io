Pubquery

Platform for pubs @ KTH

About

Pubquery is a platform that lets event managers market and manage their events in a way that allows for potential customers to simply evaluate their options on a per-day basis. Built by four students at KTH, the Pubquery eco-system spans a website for students to show open pubs, an app for pubs to manage their events, and an API providing interconnectivity.

Web

The Pubquery web, a simple website showcasing the open pubs at KTH a certain day of the week. With information such as opening hours, number of people attending and theme for pub, the website gives potetnial attendees all the necessary information for a active life on campus.

URL: pubquery.se

App

The Pubquery app enables Pubmasters to seemlessly create and manage events displayed on the website. Ones an event occurs, the counter app integrates with the website to real-time display the current ammount of people attending the pub. Built in Typescript as a PWA, cross-platform is enabled from the start allowing users on a variety of popular celular devices.

API

The Pubquery API is the software providing the interconnectivity between the app and the website. By integrating endpoints to a MySQL database it stores all the data and delivers it in a formatted way.
